## configure

create an `.npmrc` with the following content (create an app @ [apps.twitter.com](https://apps.twitter.com))

```
twitter_consumer_key=
twitter_consumer_secret=
twitter_access_token_key=
twitter_access_token_secret=
```

## start

```
npm start
```
